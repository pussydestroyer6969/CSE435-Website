Project Background

As vehicles become more advanced, software-driven features of the vehicles have taken a more active role in providing both convenience and saftey to the occupants. Hardware advancements such as radar sensors, camera and other inputs pave way for the ability for vehicles to calculate ad control the vehicle in real-time to react to the uncertainty of the roads. In this project, we will be developing a Scalable Cruise Control (SCC) system that can assist the driver to create a safe and enjoyable cruise.


Project Description

In this project, we will be developing a Scalable Cruise Control that will consist a simple cruise system, a following distance management system and automatic emergency brake system. The simple cruise system will allow the driver to set a speed to be maintained, which can be adjusted through user input and stopped with a brake action. The following distance management is a system that will build on top of the simple cruise system. It allows the user to set a distance to be kept from a leading vehicle. The distance will be maintained through real-time data collected by the vehicle's sensor. Finally the automatic emergency brake system will operate with or without the simple cruise system. A vehicle will apply braking pressure to minimize stopping distance if a driver cannot react in time to prevent an impending crash.
